# QkEngine
Simple game engine made in C++

## Features
* Component based architecture
* Grid based NavMesh
* Model import
* AABB collisions
* Audio player
* UI Image, Button
* Serialization

## Dependencies
* OpenGL
* Glad
* GLFW
* IrrKlang 1.6
* RapidJson
* ImGui
* Assimp
